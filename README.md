## 特性

- 基于 **Alipay Design** 设计规范；
- 支持多端小程序（支付宝，淘宝，钉钉等）；
- 支持主题配置切换；
<!-- - 支持 `px` 与 `rpx`； -->

## 安装

```bash
$ npm install evil-ui --save
```


## 使用

在remax中使用

```js
import {View,Input} from 'evil-ui'
```
